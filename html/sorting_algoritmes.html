Bubble sort

Bubble sort is a very simple sorting algorithm, the drawback of bubble sort is that It is quit slow for larger data sets. It works by going through an array and swapping and swapping the adjunct elements, if they are in the wrong order.

The algorithm:

  • 1. This sorting algorithm works by doing multiple passes through an array.

  • 2. Compare the current element whit the next one

    • If they are not in the correct order, swap them

    • Move to the next pair of elements

  • 3.Repeat step 2 until you reach the end of the array

  • 4.Repeat step 1-3 but each time the last sorted element. (since it's already in the correct position).

  • 5. Continue until you make a full pass without any swaps, at that point, the array is sorted.

Bogo sort

Bogosort, also known as stupid sort, is a sorting algorithm that was created as a joke. It is the most inefficient sorting algorithm that exist. However, the running joke is that it could theoretically be the best sorting algorithm, if it happens to sort a list correctly in one random shuffle. In practice it usually takes a lot of time and most likely never finish sorting a list. This is because the algorithm works by shuffling the array and checking if the array is sorted.

The algorithm:

  • 1. Shuffle the array in a random order

  • 2. Check if the array is sorted

    • No: repeat stap 1

    • Yes: the array is sorted

selection sort

Selection sort is another very simple sorting algorithm that works by repeatedly selecting the smallest element and placing it at the front of the array. You can visualize it like sorting a deck of cards using two groups: the sorted cards and the unsorted cards. Each time you find the smallest card in the unsorted group, you move it to the sorted pile. The only difference is that, in code, we use one array instead of two separate groups.

The algorithm:

  • 1. This sorting algorithm works by repeatedly selecting the smallest element from the unsorted part of the array.

  • 2. Start at the first index.

    • Search through the rest of the array to find the smallest element.

    • Swap it with the element at the current index.

  • 3. Move to the next index and repeat step 2.

  • 4. Continue until the whole array is sorted.
